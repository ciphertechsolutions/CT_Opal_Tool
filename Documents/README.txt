The documents in this directory come from the Trusted Computing Group (TCG)
web site at www.trustedcomputinggroup.org.

These documents are intended to be used along with the CT Opal Tool for
developing software for Opal hard drives.  In addition, the tools and
documents can be used to explorer the properties of Opal hard drives
to give the developer a better understanding of how Opal drives can
be fully utilized.
